# hitokoto

在终端中获取[一言](https://hitokoto.cn)

## 依赖

### Debian/Ubuntu

```bash
sudo apt install libcurl4-openssl-dev libjson-c-dev
```

### macOS

```bash
brew install curl json-c
```

## 编译

```bash
git clone https://github.com/moesnow/hitokoto.git
cd hitokoto
make
```

## 用法

```bash
./hitokoto
```
